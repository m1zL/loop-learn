## 実装メモ

- Credentials Provider はパスワードを**自動でハッシュ化しない**。`authorize` 関数内で `bcryptjs` 等を使って手動でハッシュ化・照合する必要がある (参考: https://authjs.dev/getting-started/authentication/credentials)
- next-pwa は next.config.ts に設定を追加するだけで動作するが、Vercel へのデプロイ時の動作確認が必要
- Playwright は `@playwright/test` をインストールし、`npx playwright install` でブラウザバイナリをダウンロード
- GitHub Actions の E2E ジョブは DB が必要なため、Supabase CLI のローカルスタックか、テスト用 Supabase プロジェクトを用意する
