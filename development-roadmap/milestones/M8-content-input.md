# M8: コンテンツ入力強化 — コーネルノート連携 & 学習リマインダー通知

> 状態: ⬜ 未着手
> 対象期間: 目安 1〜2週間
> 依存: M5完了 (AIカード生成基盤が必要)

---

## 概要

カード作成の摩擦をさらに下げる2つの機能を追加する。
コーネルメソッドノート連携では既存のノートフォーマットをそのままカード化できる。
学習リマインダーでは「復習忘れ」を防ぎ、継続率を高める。

## 実装メモ

- Vercel Cron Jobs は `vercel.json` の `crons` フィールドで設定。無料枠は1日1回まで
- Web Push は iOS Safari 16.4+ からサポート。それ以前のユーザーはメール通知のみ提供
- コーネル形式の3エリアは `<textarea>` を3つ並べるだけのシンプル実装で十分
- プッシュ購読は `PushSubscription` として DB の `User` テーブルに JSON カラムで保存する (マイグレーション追加が必要)

---

## リスク

| リスク | 影響度 | 対策 |
|--------|--------|------|
| Vercel Cron 無料枠 (1日1回) が不足 | 低 | 1回/日のリマインダーなら無料枠で十分 |
| Web Push の VAPID キー管理 | 中 | Vercel 環境変数で安全に管理し、コードに含めない |
| メール配信率の低下 (迷惑メール判定) | 中 | Resend の送信ドメイン認証 (SPF/DKIM) を設定する |
